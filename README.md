## Parallel QuickSort
/parallel folder contains:

  * mainSLURM.slurm : compiles and runs the code for 1M, 2M, 4M and 16M  of instances

* /include/methods.h, contains the fuctions shared between 'main.c' and 'mainMPI.c'
